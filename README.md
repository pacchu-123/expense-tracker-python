
#  Expense Tracker with Visuals (Python)

##  Introduction

This project helps users track daily expenses and analyze spending patterns using Python.

##  Features

* Track expenses (date, category, amount)
* Category-wise analysis
* Bar chart visualization
* Pie chart visualization
* Budget alert system
* Export data to Excel

##  Technologies Used

* Python
* Pandas
* Matplotlib
* Excel

##  Output

* Expense by category (Bar Graph)
* Expense distribution (Pie Chart)

##  How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the program:

```
python expense_tracker.py
```

##  Dataset

The dataset contains:

* Date
* Category (Food, Travel, Shopping)
* Amount

##  Conclusion

This project helps understand spending habits and improve financial planning.
